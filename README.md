# MCP Leading the People

Interactive presentation for the [Claude Code Paris](https://luma.com/zdk7lr94?tk=A0f27d) meetup — February 2025.

**[View the presentation](https://benoitvx.github.io/claude-code-paris-02/)**

## About

A talk about connecting AI to French public data through the MCP (Model Context Protocol) Gateway. Covers the problem of LLM hallucinations on public data, the architecture of a government-backed MCP gateway, and real use cases with authoritative sources like Legifrance, data.gouv.fr, and service-public.fr.

## Tech Stack

- Next.js 14 (App Router, static export)
- TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on GitHub Pages

## Run locally

```bash
npm install
npm run dev
```
